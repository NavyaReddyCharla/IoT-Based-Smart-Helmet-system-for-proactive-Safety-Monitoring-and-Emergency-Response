# IoT Based Smart Helmet System for Proactive Safety Monitoring and Emergency Response

An IoT-based smart helmet designed to improve two-wheeler rider safety by ensuring helmet usage, detecting alcohol consumption, and providing automatic accident alerts with real-time location tracking.
